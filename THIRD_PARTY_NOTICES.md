# Third-Party Notices

This file lists projects and tools that influenced TXTeditor or whose behavior was used as a reference.

## d2rlint

TXTeditor's D2R lint rules are based on the behavior of d2rlint by eezstreet. Portions of the lint logic have been ported or adapted for TXTeditor's live Problems panel diagnostics.

d2rlint states that it is licensed under the GNU GPLv3 license. Because TXTeditor includes d2rlint-derived lint behavior, TXTeditor is distributed under the GNU General Public License v3.0 or later.

- Project: d2rlint by eezstreet
- License: GNU GPLv3

## D2ExcelPlus

D2ExcelPlus is credited as inspiration and reference for practical Diablo II / Diablo II: Resurrected table editing workflows. I found it to be an excellent and very stable program, personally ported and used it in my own workflow, and that experience helped motivate TXTeditor as a small personal editor project.

The D2ExcelPlus license text provided by the user is MIT-style. TXTeditor credits D2ExcelPlus here as an important inspiration and reference.

## AFJSheet

AFJSheet is credited as inspiration for a long-standing Diablo II table editor workflow and for helping shape what a practical TXT editing workflow can feel like.

## OpenAI Codex

Most implementation work on TXTeditor was produced through collaboration with OpenAI Codex. OpenAI Codex is credited as development assistance and is not a runtime code dependency of TXTeditor.
